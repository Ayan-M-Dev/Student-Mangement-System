# Blockchain Service (Hardhat)

This service manages Certificate logic using a local Hardhat node.

## Features
- **Smart Contract**: `Certificate.sol` for tamper-proof records.
- **Scripts**: Automated deployment and issuance.
- **Frontend Integration**: Fully integrated with the App.

## Running
This service runs automatically as part of the main stack.
