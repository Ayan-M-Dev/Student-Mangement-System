# School Management System - Frontend

This is the React/Vite frontend application.

## Key Features
- **Student Management**: CRUD operations.
- **Notices**: Announcement board.
- **Blockchain Integration**: 
    - **Login**: "Connect Wallet" button.
    - **Certificates**: Issue/Verify certificates directly from Student Profile.

## Running
This service runs automatically as part of the main stack.
Please see `INSTRUCTIONS.md` at the project root for verification steps.
